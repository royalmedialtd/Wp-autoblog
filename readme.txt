Discover trending topics in SEO, Marketing, Web Development, and Mobile Development, then generate structured blog posts with one click.

== Description ==

TechNews Autoblog helps you discover trending topics and create structured blog post drafts automatically. The plugin fetches trending news from:

* **NewsAPI** - Real-time tech news headlines
* **Google Trends RSS** - Daily trending searches

### Features

* **Trending Dashboard** - View current trending topics in your niches
* **Category Filtering** - Filter by SEO, Marketing, Web Development, or Mobile Development
* **One-Click Posts** - Create structured draft posts from any trending topic
* **AI Integration** - Optional OpenAI integration for AI-generated content
* **SEO Ready** - Integrates with RankMath, Yoast SEO, and All in One SEO

== Installation ==

1. Upload the `rms-autoblog` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **TechNews Autoblog > Settings** and enter your NewsAPI key
4. Visit the **TechNews Autoblog** dashboard to start discovering trends!

== Getting API Keys ==

### NewsAPI (Required - Free)
1. Visit [newsapi.org](https://newsapi.org/register)
2. Create a free account
3. Copy your API key
4. Paste it in **TechNews Autoblog > Settings**

Note: Free tier allows 100 requests per day.

### OpenAI (Optional - Paid)
1. Visit [platform.openai.com](https://platform.openai.com/api-keys)
2. Create an API key
3. Paste it in Settings to enable AI-generated content

== Frequently Asked Questions ==

= Is this plugin free? =
Yes! The plugin is completely free. You just need a free NewsAPI key (100 requests/day).

= Can I generate full blog posts automatically? =
By default, the plugin creates structured outlines. Add an OpenAI API key to generate complete draft content.

= What categories are supported? =
SEO, Marketing, Web Development, and Mobile Development.
